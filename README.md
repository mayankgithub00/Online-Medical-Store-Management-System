# Online-Medical-Store-Management-System
Managed And Implemented The database through SQL for a Medical Store.


Managed And Implemented The database through SQL, where customer can place a order, make a transaction, see the status of the order, stocks available, track shipment and seller can also see the status of consumers, orders and can track the various shipments.

Used the core concepts of DBMS such as Indexes, implemented various SQL Queries , Packages, PLSQL, procedures, sequences etc.

Tech stack used: Oracle Apex Online Server, SQL.
